## How to Run the Project

1. Download the model file from the Releases section:

   * Go to "Releases" in this repository
   * Download `fusion_model_best.pth`

2. Place the file in the same folder as the notebook:

```
plumbago/
├── hybrid-deep-learning-framework-for-plumbago.ipynb
├── fusion_model_best.pth
```

3. Open the notebook and run all cells.

The model will load and the system will work correctly.
